### 类Unix系统  Linux/macOs

#### 信息类：
ls -alh 全部信息

ss -tuln （linux
lsof -i port  -p pid （mac

ps aux 瞬时进程
top 实时进程
pgrep chrome   返回pid

tail log.txt 跟踪查看文件末尾情况

history NUMBER

ls -l  查看权限
-rwxrwxrwx 第一个是-，代表是文件(eg: drwxr--r-x)，是d，代表是目录，然后剩下九个分为三组，对应所有者，所有组，其他组，对于这个文件/目录的权限
r代表4
w代表2
x代表1
chmod 777 代表对于每个组的每个权限都开放
stat FILENAME 查看特定文件权限

#### 操作类：
kill -9 < PID >

pkill chrome 按照进程名，关掉所有chrome

touch file 创建文件

mv 移动/重命名
